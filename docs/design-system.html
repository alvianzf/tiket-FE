01 — Foundations

Color System

Inconsistency resolved: The Tailwind config defined primary: #3C9DFF but the actual Navbar, the CSS variable --primary, and the .bg-blue utility all use #4267B2. Since #4267B2 is referenced more across the codebase and matches the logo, it is the canonical primary. Similarly, two near-identical dark values (#2F3033 vs #2B2D31) and two cyan values (#0AD1FF vs #00D5FF) were resolved to the Tailwind config values.
